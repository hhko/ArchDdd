# 아키텍처에서 인터페이스 역할은?

인터페이스는 사용자와 구현자 관점으로 분리 시킬 수 있기 때문에 **레이어를 연결시키는 역할을 책임집니다.** 
```
기술    -{입력 호출}->  유스케이스  -{출력 호출}->  기술
사용자                  구현자
                        사용자                    구현자
```

## 기술이 비즈니스를 호출할 때
```
기술 -{입력 호출}-> 유스케이스
```

## 비즈니스가 기술을 호출할 때
```
유스케이스 -{출력 호출}-> 기술
```
