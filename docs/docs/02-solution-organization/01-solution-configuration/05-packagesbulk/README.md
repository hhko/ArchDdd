# 패키지 Bulk 추가/제거하기

![](./img/2024-03-15-23-31-15.png)


```shell
Packages-{Feature}.txt      # 패키지 목록
Packages-{Feature}.bat      # Tests와 Src 대상 프로젝트을 구분한 param을 ps1에 전달하여 실행한다
Packages-{Feature}.ps1      # Bulk 패키지 추가/제거한다
```
- Feature: `Add`, `Remove`
