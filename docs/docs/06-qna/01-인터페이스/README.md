# 인터페이스

1. 아키텍처에서 인터페이스 역할은?
   - 인터페이스는 사용자와 제공자로 분리 시킬 수 있기 대문에 비즈니스 관심사 레이어와 기술 레이어를 느슨하게 연결 시킵니다.
     ```
     - 인터페이스 사용자: 인터페이스 정의을 사용한다.
     - 인터페이스 제공자: 인터페이스 구현을 제공한다.
     ```
