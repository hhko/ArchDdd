# FAQ

## 경계
1. 인터페이스 이름은?
   - 인터페이스 이름은 사용자 관점으로 작명하고 인터페이스 구현은 제공자 관점에서 개발한다.
     ```
     - 인터페이스 사용자: 인터페이스 정의을 사용한다.
     - 인터페이스 제공자: 인터페이스 구현을 제공한다.
     ```
1. 클린 아키텍처에서 중재자 패턴 역할은?
1. 중재자 패턴 사용시 클린 아키텍처에서 인터페이스 역할은?
